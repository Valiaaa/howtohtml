+ 开始写简单的文档吧

- 开始写简单的文档吧

一个html文档基本由四个部分组成：head, style, body, 和script。其中，style可以用后缀为css的文档代替，而script则可以用后缀为js的文档来代替。
      * 如果你选择用jQuery写js的话，记得把js文档放在jQuery link的下方，不然它不work
code里的每一个部分都是由<部分名></部分名>开始与结束。
html文档里的所有内容都要写在<!DOCTYPE html></html>之中。
随后，就是由<head></head>开始与结束的head部分。
head里会包括：基础浏览器适配设置，链接（比如网站自定义的小图标，和css文件），script（比如jQuery版本网址，js文档），以及网页自定义标题。
      * 比如，本网页head里的内容是：


<meta charset="UTF-8">
<meta http-equiv="X-UA-Compatible" content="IE=edge,chrome=1">
<meta name="description" content="">
<meta name="viewport" content="width=device-width">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<link rel="icon" href="icon.png">
<link rel="stylesheet" type="text/css" href="style.css">
<script src="https://code.jquery.com/jquery-3.5.0.js"></script>
<script src="script.js"></script>
<title>How To Html</title>

其中，需要自行修改的部分是icon的link以及网页的title。

随后就是最为主要的body部分。这里囊括了你想在网页里展示的所有内容。其中，你可以用很多种元素来定义一段内容（的视觉形式）。
      * 你可以在css里分别定义这些元素的属性，来让他们拥有不同的eg.颜色，字体，字号，对其，hover效果等
      * 比较常用的元素有：p（paragraph，代表普通正文）；h1-6（正文之外的特殊文字格式，如果你需要更多格式还可以自己额外用“.编号”来创建）；div（区域，你可以把很多不同格式的文字都放在这一个区域里面，然后对于整个区域进行class或者id的编辑。这里需要注意的是，div默认是新的区域，所以会换行，并且会受默认margin和padding的影响）；span（没有实质性意义，也不会让范围内的内容和前后文分离，适用于你想在一段连贯的文字里让几个特殊的文字有特别的css）。
      * 几段文字之间的回车并不会被coding软件识别，所以需要用<br>来手动分割一段文字。类似的，大于一个的空格也不会被多次识别，所以需要用&nbsp来代替剩余的空格。

练习：更改网页的名称，并用至少两种文字格式在网页上写下自定义内容。并且其中一种格式的文字需要超过两个自定义分行。

如果想要在网页中加入图片，我们首先需要将图片转移到code的本地文件夹中。如果图片较少可以直接放在文件夹里，如果比较多建议新建文件夹来给图片归档。
直接使用<img src="图片名+后缀">就可以置入图片了。
      * 如果图片在子文件夹中，则需要写子文件夹名/图片名+后缀。
      * 如果图片和html文件在不同的文件夹，但是共处一个母文件夹，则需要写../图片文件夹名/图片名+后缀。
      * 在定义完图片路径后，一样可以给图片加上class或者id，或者直接加上style来设置规格（比如如果这张照片只需要设定宽度，并且这个设定和其他所有内容都不重复，那么可以直接用style；如果有多个内容都会需要一样的style，那么建议给它们都加上同一个class，然后在css文档里统一规定）。
如果想要在网页中加入超链接，可以直接使用<a href="https://目标网址">超链接替代文字</a>。
      * 然而，如果这么link，该目标网址会直接在同一个tab打开，这意味着它会覆盖原来的网页。如果想要目标网址在新标签页里打开，那么需要在“目标网址”后加上target="_blank"。
      * 这个link功能不只能链接网页，还能够链接你所创建的其他html文件。你只需要在href=后面写上“html的路径”即可。

练习：在刚刚的网页中添加至少一张图片，和至少一个超链接。

+ 用css更改视觉规则

- 用css更改视觉规则

你可以使用并自定义h1-h6来代表特殊格式的文字，主体的text可以用body或是p(paragraph)来定义。
      * 在css文档中先写出你想要编辑的组名，例如h1，随后使用{}来框住你定义的部分。
      * 每个定义之后一定要记得加上；。
你可以使用并自定义*来定义整个网页统一的格式，例如margin和display。
一些常用的css编辑项为：


background-color：背景颜色，可以用预设颜色（eg.red）、rgb（eg.rgb(0，0，0)）或者hex color code（eg.#CDCDCD）来编辑。
color：本体颜色，同上。
opacity：透明度，从0.0-1.0。可以直接加在bg-color之后，也可以使用rgba来编辑（eg.rgba(0, 128, 0, 0.3)）
* 这些颜色都可以在：后使用linear-gradient的指令形成渐变色，eg.background: linear-gradient(to bottom, #33ccff 0%, #ff99cc 100%)。方向有上下左右以及斜上下左右。

练习：更改网页的背景颜色为渐变色，并在上面写一行有颜色的文字。

border-style：边框。none/hidden/solid/double/dotted/dashed/groove内浮雕/ridge外浮雕/inset内凹/outset外凸。还可以mix边框，eg.border-style: dotted dashed solid double，出现的顺序是上右下左。
border-width：边框粗细，可以用px等单位定义，也可以用thin，medium和thick定义。同样的，如果定义两个数值，那么前一个数值是上下，后一个数值是左右；如果定义四个数值，那么顺序依旧是上右下左。
border-color：边框颜色，同上面的颜色介绍，也同样可以定义四个数值。
border-radius：边框圆角，可以用px等单位规定。
* 如果只想要一边有边框，那么可以使用border-top/bottom/left/right来实现。
* 如此，就可以用border-bottom来设定分割线。
* 也可以把border-width，border-style (required)，border-color同时写在border指令里。eg.border: 5px solid red。

练习：添加一行有边框的文字，且边框含有两个样式。

margin：元素周围的空间，在所定义的border之外。auto让浏览器自动定义/length使用类似px的单位/%元素范围内百分比/vw整个屏幕的百分比/inherit继承母元素的数值。这里的数值可以定义为负数。
* 可以用margin-上下左右分别定义长度。
* 也可以在同一个指令里定义4/3/2个数值，分别是上、右、下、左/上、左右、下/上下、左右。
padding：元素周围的空间，在所定义的border之内。特殊设定同上。
height/width：元素长度宽度，可以用上面介绍过的不同方式定义。
* 假设你设置了300的宽度和25的padding，那么这个元素的实际宽度会是350。如果你希望整个元素的宽度依旧为300的话，可以使用box-sizing: border-box来把整个内容框在一个框里。
max-height/width：最大长宽。如果你设置了一个特别大的width，那么当浏览器宽度变窄的时候框内的内容可能会溢出；而如果你设置的是max- width，那么它会根据浏览器的宽度自动缩短，避免用户需要横向滑动页面。

练习：设置有边框的文字的margin和padding，将它的最大宽度设置为百分之六十。

font-family：字体，可以用浏览器支持的内置字体，也可以导入自定义字体。格式为：字体名称，serif/sans-serif。
font-size：字体大小，可以用常用单位定义。
line-height：行间距，同样用常用单位定义。
text-align：文字排列，可以设置left/center/right/justify每行宽度相同。但是justify并不常用于传统html的设计中，基本左中右就够用。

练习：将边框内的文字设为居中排列，并更改字号和行间距。

你可以通过例如Google Font/Adobe Font/本地字体等方式来导入自定义字体。
如果想要导入本地字体的话，可以先把字体复制到本地的文件夹内，然后在css文件中输入：

@font-face {
font-family: '自拟字体名称';
src: url('本地字体路径');
}

或者，在Google Font中，点进你想要的字体，在Style板块中确认你想要的字体粗细，并点击右边的加号。随后，在右上角的Selected Family页面中点击@import的选项，将下面的代码黏贴进css文档里，并根据给定的字体名称来选用。
亦或者，在Adobe Font中（需要登录），点进你想要的字体，点击Add to Web Project，新建或者选择已有project，选择你想要的粗细，点击save，就可以看到导入code了。最后根据给定字体名称使用。

练习：导入一个自定义字体并使用。

+ 将本地网页上传到网络上吧

- 将本地网页上传到网络上吧

点开GitHub Desktop的页面
如果左下角的Summery写了required（一般你进行了一些大更新后就会需要你概括一下你做了啥），那么你可以随便打个啥敷衍了事，也可以用几个词大致介绍一下你的更新。
      * 最新的Summery会显示在你GitHub coding页面的文件旁边，所以可以稍微写一下。
点击蓝了的Commit to Main
在右边，你会看到app提示你Publish Branch，点击它。这代表着你在本地的变动已经被上传到了你之前创建的repository里。
      * 在之后的更新中，在相同位置点击Push Origin就可以自动update你的网页了。
回到你的repository页面，刷新，你就能看到你所上传的所有文档了。
点击菜单中的Settings，在左侧点击Pages，然后将Branch的None改为Main，点击Save。
稍等一会儿，刷新页面，你就能够在该页面中看到你网站的网址啦。
完成！所有人都可以通过网络看到你的网站啦：）
